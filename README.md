# Hi, I'm Ellie 👋

I'm a developer with a strong eye for design — I build complete products from concept to deployment, handling both the technical and visual side when needed.

Currently focused on full-stack web development with React and Node.js, while also working on a 2D pixel art game on the side.

---

## 🔹 What I do

- **Full-stack web development** — building and deploying complete applications with React, Node.js and MongoDB
- **UI design** — designing interfaces with attention to detail, because good code deserves good visuals
- **Game development** — designing and coding a 2D pixel art game in Java
- **Pixel art** — creating game assets and illustrations with Aseprite and Pixquare

---

## 🚀 Projects

**[elliecreations.it](https://elliecreations.it)** — my personal portfolio, currently being rebuilt from PHP/CSS to a modern React/Vite + Node.js + MongoDB stack.

**[Italy Selected](https://italyselected.com)** — a luxury travel platform for curated Italian experiences across multiple destinations.
`React/Vite` `Node.js/Express` `MongoDB` `Tailwind CSS` `DigitalOcean Spaces`

**2D Pixel Game** — a Java-based 2D game with original pixel art, actively in development.
